Rigging in animation is a technique used by animators toquite literallyrig up a statistic digital character to act or move.

The rig in this case, is usually made as a sort of one-size-fits-all kind of tool, which you can use in order to create pre-made skeletal animations that replicate motions like walking, running, jumping, etc., and apply onto static characters.
